## Step 9 - layers, factory, CLI (next up)

1. **`SandboxLayer`** (`_async/layers.py`, codegen-able - no SDK imports):
   wraps any backend, same `StorageBackend` protocol. One choke point:
   prefix every incoming path with `root`, validate the (already
   normalized) result stays under it; reverse-strip where paths surface.
   Replaces `sandbox.py` + all `if self._sandbox` branches. `Storix(b,
   sandboxed=...)` sugar optional. Symlink escape inside LocalBackend
   base is a documented non-goal of lexical sandboxing (parking lot).
2. **`storix.temporary()`**: sync+async context manager yielding
   `Storix(LocalBackend(mkdtemp()))`, deleting the tree on exit. Lives
   in the flavor trees, re-exported.
3. **Factory + config rework** (owner-flagged as sloppy; redesign, don't
   port). Sketch to refine together:
   - per-backend config models (pydantic-settings) with standardized
     prefixes: `STORIX_PROVIDER`, `STORIX_LOCAL_BASE`,
     `STORIX_AZURE_CONTAINER|ACCOUNT_NAME|CREDENTIAL` - retiring the
     flat `ADLSG2_*`/`STORAGE_*` blob (keep aliases one release?).
   - each backend ships `Config` + `from_config(cfg)` classmethod; a
     tiny name->backend registry; `get_storage(provider=None,
     **overrides) -> Storix` = settings -> config -> backend -> core.
   - open questions for the brainstorm: classmethod shortcuts
     (`Storix.local(...)`, `Storix.azure(...)`)? URL-scheme factory
     stays 0.3.0 (entry-point registry, see deferred doc).
4. **CLI rewire**: `cli/` consumes the sync `Storix`; behavior-preserving
   swap of construction + error handling (typed errors now raise).

## Design sketches agreed in-chat (not yet built)

- **URLs**: `fs.data_url(path)` core-generic (cat+base64, all backends);
  `fs.url(path, *, expires_in)` -> new port method with
  `UnsupportedOperationError`-raising default on `BackendBase`, gated by
  `capabilities.presigned_urls`; Azure implements SAS (absorb legacy
  `security.py`). Non-breaking port growth via the ABC default.
- **Metadata**: `RawStat.metadata: Mapping[str, str] | None = None`
  (kw-only, non-breaking); `write`/`echo` gain gated `metadata=` under
  `capabilities.custom_metadata` (Azure native; local later via the
  sidecar `MetadataLayer` in deferred-decisions.md).

## Parking lot (conformance decisions pending)

`delete('/')` / `rm -r` of cwd or root (azure refuses via
`UnsupportedOperationError`, memory/local currently allow); stale yields
under concurrent deletes; `mv` of a path into itself; symlink escape in
LocalBackend; container auto-creation (AzureBackend requires existing).
